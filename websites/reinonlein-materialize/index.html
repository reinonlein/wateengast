DATA SCIENCE

Wat je hier allemaal voor moois kunt verwachten? Nou, bijvoorbeeld de leukste data science-resultaten. Data Science is het mooiste vak dat er is, zeker als je het toepast op de meest alledaagse dingen. Ik ben van plan om de wonderlijkste technieken te gaan toepassen op de droogste onderzoeksvragen die je maar kunt verzinnen.

DATA VISUALISAITE

Ik heb een zwak voor gave data visualisaties en ik ben erachter gekomen dat er wel zó ontzettend veel mogelijk is op dit gebied met de technologieën van vandaag. Hier ben ik van plan om een mooie showcase bij te gaan houden. Onder meer van mijn experimentjes in D3.js, de mooiste visualisatie library die ik ken!

WEB DEVELOPMENT

Mijn nieuwste hobby is het handmatig in elkaar programmeren voor websites. Voor op internet. Normaal gesproken deed ik dit alijd in Wordpress, maar door zelf in de code te duiken gaat plotseling de hele wereld voor je open. Alles wat je kunt verzinnen, kun je bouwen in Javascript, HTML en een fraai CSS framework zoals bijvoorbeeld Bootstrap of Materialize. Hier lees je straks mijn ervaringen en resultaten.

Stay tuned!
