How It's Made

This site is made by me, myself, in pure joy and entertainment. This site is my little happy place online, where I can bring my frontend coding skills to practice and create an engaging portfolio-site on the go. The tools I used to make this site are ofcourse my favourite coding text editor: VS Code. Using the Live Server extension, it's really easy to type HTML, CSS & JavaScript and check the results in real time.

This site also makes use of the amazing webframework Materialize by Google. It had been a close call between Bootstrap 4 (also amazing!) and Materialize, but in the end some nice visual features of Materialize flipped the coin in their direction. Nonetheless: Bootstrap has won my heart as well, mainly for the broad range of features and the super-usefull margin and padding style classes.

Finally, for the backend of this site I use Firebase, also by Google. This is another amazing (free) product which makes your geeky life much more fun. The main features I use are the realtime NoSQL database Firestore (for my visualisations) and their Authorisation features. But Firebase has so much more to offer! Finally, I really believe I can create anything online now. The sky is not the limit! Only spare time is...
